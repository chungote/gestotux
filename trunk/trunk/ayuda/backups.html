Backups

¿Que utilidad tienen los Backups?

La principal utilidad de los backups es el resguardo seguro de sus datos ante la posibilidad de que se puedan llegar a perder ante la falla del programa, falla del equipo donde esta trabajando, etc...
Esto le va a permitir recuperar sus datos y poder seguir utilizando los datos y el programa ante cualquiera de esas fallas.

¿Que frecuencia deben tener los backups?

Aunque la frecuencia es muy dependiente del tipo de trabajo que realize, un backup semanal o diario si el movimiento de datos de su empresa es muy grande es aconsejable.
Si el ingreso de datos es mas esporadico, cada 15 o 30 dias es aconsejable realizarlos. Recuerde que si sucede una falla que le prive de sus datos, tendra los datos hasta al aultima fecha que ralizo el ultimo backup del sistema.

¿Como se realizan?

Para realizar un backup debe dirijirse al menu de Herramientas y luego a la opcion Backup que esta al final del menu.

Esto abrira una ventana que le permitira realizar backups y restaurarlos tambien.

Crear un backup

Existen basicamente 3 opciones que puede elegir al crear el backup:

  • Guardar la configuración del programa
  • Guardar la estructura de las tablas
  • Guardar los datos del programa

Por supuesto, la ultima opcion es la que mas nos interesa, y es la que esta predefinidamente seleccionada. La opción de guardar la estructura de las base de datos es para cuando se desea transportar la base de datos completa a otra copia del porgrama que nunca haya sido utilizada; y si a esto le agregamos la configuración del programa se puede cambiar de pc el sistema sin modificaciones en las preferencias.

Para iniciar el backup presione el boton de Iniciar y vera en la barra de progreso como avanza el backup.
Al completarse, se abrira una ventana de dialogo que le pedira el lugar donde desea colocar el archivo generado como backup. Este archivo es importatne y es el que usted debe guardar en un lugar distinto a su pc, ya sea grabandolo en un cd, un pen drive o llevandolo a otra maquina.

Restaurar un backup

Los pasos para restaurar un backup son sencillos.
Seleccionando la pestaña de Restaurar Backup aparecera un boton que nos permite buscar el archivo de backup que tengamos guardado.

Una vez ubicado y abierto el archivo, aparecera en el cuadro de al lado. Simplemente presionamos Iniciar y se comenzara la restauracion de los datos contenidos en el backup.

Se aconseja reiniciar el programa luego de restaurar un backup para la correcta inicializacion de la base de datos recien restaurada.
